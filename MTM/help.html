Editing Time Tables

Best viewed in Firefox, Chrome, etc.

Creating & Deleting Time Tables

To Create a Time Table...

  1. Under the "Date/Name" heading, choose the date name style that will appear in the table header.
    • Long date (Wed Nov 21 2018) default
    • Short date (11/21/2018)
    Note: Tables can not be created without names!
  2. Click the button to continue.
  3. A dialog box will open allowing the user to modify the name of the table.
  4. Click "OK" to create the table.

To Modify a Table's Name after Creation...

  1. Double click the table's "Name"
  2. A dialog box will open allowing user to modify the table's name.
  3. Click "OK" to continue.

To Remove a Time Table...

  1. Click the ✖ in the upper right corner.

Adding, Inserting & Deleting Time Lines

Note: All new lines added or inserted will display the current time in the "Start" cell, and the current time plus 2 minutes in the "Stop" cell.

To add a Time line at the top of a table...

  1. Click the ➕ symbol to the left of the "Description" column header.
  2. Modify as needed.

To insert a new line under an existing line...

  1. Double click any "Hrs" cell to insert a new time line below the level of the clicked cell.
  2. Modify as needed.

To delete a Time line from a table...

  1. Click the checkmark (✓) at the left of any time line to delete it.

Editing Time Lines

Double click any "Description" cell to open a dialog box that allows user input.

Double click any "Start" cell to insert the current values of the "Start Time" and "Stop Time" fields into their coresponding cells.

Double click any "Stop" cell to insert the current value of the "Stop Time" field into it.
